# Mi Portal Digital

Portal web profesional que combina servicios de desarrollo, venta de software, blog, herramientas interactivas y sistema de suscripcion.

## Tecnologias

- Jekyll 4.x (github-pages) - Generador estatico
- Tailwind CSS v4 - Estilos utilitarios
- Alpine.js 3.15 - Interactividad frontend
- Font Awesome 7 - Iconografia
- marked.js + DOMPurify - Parseo Markdown seguro
- JSZip + FileSaver - Generacion de archivos ZIP

## Instalacion local

git clone https://github.com/tu-usuario/tu-usuario.github.io.git
cd tu-usuario.github.io
bundle install
npm install
npm start

## Despliegue

El sitio se despliega automaticamente en GitHub Pages al hacer push a main.
