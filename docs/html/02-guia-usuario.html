Sómo comprar software

  1. Ve a Software o Dev Toolkit.
  2. Haz clic en "Buy Now".
  3. Será redirigido a Gumroad para completar el pago seguro.
  4. Recibirás el producto por correo electrónico.

Sómo usar las herramientas

Cada herramienta tiene instrucciones en pantalla. Por ejemplo, el Conversor Markdown permite pegar texto, ver la vista previa en tiempo real, copiar el HTML y descargarlo como archivo.

El Generador C++ permite seleccionar tipo de proyecto, estándar C++ y opciones adicionales, luego descarga un ZIP con la estructura lista.

Suscripción al newsletter

En el footer, en el blog y en la página Free Pack puedes suscribirte. Recibirás el C++ Booster Pack gratuito y una secuencia de emails con contenido exclusivo.

Contacto
