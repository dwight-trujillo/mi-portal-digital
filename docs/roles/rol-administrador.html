🗥️ Documentación para Administradores

Configuración inicial

  1. Configurar variables en _config.yml.
  2. Reemplazar IDs de Formspree y Gumroad.
  3. Configurar Cloudflare si se usa dominio.

Desplegue

El despliegue es automático al hacer push a main. No se requiere intervención manual.

Monitoreo

  • UptimeRobot: configurar monitor HTTP(s) cada 5 min.
  • Sentry: para errores JS en cliente (opcional).
  • GitHub Security: revisar alertas de Dependabot.

Backups

Se realiza mirror automático a GitLab mediante workflow. Mantener una copia local.
