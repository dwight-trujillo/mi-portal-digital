Entorno de desarrollo

git clone https://github.com/tu-usuario/tu-usuario.github.io.git
cd tu-usuario.github.io
Instalar dependencias
bundle install
npm install
npm start

Estructura del proyecto

mi-portal-digital/
├── .github/workflows/      # CI/CD y seguridad
├── _includes/               # Componentes HTML reutilizables
├── _layouts/               # Plantillas Jekyll
├── _posts/                  # Artículos
├── assets/                  # CSS, JS, imágenes
├── herramientas/            # Páginas de herramientas
├── _config.yml              # Configuración
└── index.html                # Landing page

Añadir una nueva herramienta

  1. Crear herramientas/nueva-herramienta.md.
  2. Crear componente Alpine en assets/js/components/nueva-herramienta.js.
  3. Importar en assets/js/main.js.
  4. Listar en herramientas.md.

Seguridad en el código

  • Todo HTML dinámico se sanitiza con DOMPurify.
  • Formularios incluyen token CSRF y campo honeypot.
  • CSP restrictiva definida en _includes/head.html.
